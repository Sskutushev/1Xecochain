# ПРОМПТ ДЛЯ АДАПТИВНОЙ ВЕРСТКИ С МЕДИА-ЗАПРОСАМИ

## КОНТЕКСТ ПРОЕКТА

Ты работаешь с многостраничным веб-приложением ECOCHAIN на HTML+CSS+JS. У проекта есть десктопная версия (1920px), и тебе нужно добавить адаптивность через CSS медиа-запросы и формулы `clamp()` / `calc()` для плавного масштабирования элементов.

```css

  width: clamp(320px, 100%, 401px);
  padding: clamp(20px, 5vw, 32px);
  font-size: clamp(14px, 2vw, 16px);
```

## ИСПОЛЬЗОВАНИЕ CLAMP() И CALC()

### Что такое `clamp()`?
`clamp(минимум, предпочтительное, максимум)` - функция для плавного масштабирования.

**Примеры:**

```css
/* Шрифты */
font-size: clamp(14px, 2vw, 18px);
/* От 14px до 18px, масштабируется через 2% от viewport width */

/* Отступы */
padding: clamp(10px, 2vw, 32px);
/* От 10px до 32px */

/* Ширина контейнера */
width: clamp(350px, 90vw, 1295px);
/* От 350px до 1295px */

/* Gap между элементами */
gap: clamp(15px, 3vw, 46px);
/* От 15px до 46px */
```

### Что такое `calc()`?
`calc()` - математические вычисления в CSS.

**Примеры:**

```css
/* Ширина с вычетом gap */
width: calc((100% - 46px) / 2);
/* 2 колонки с gap 46px */

/* Высота с вычетом header */
height: calc(100vh - 80px);
/* Viewport height минус header */

/* Padding динамический */
padding: calc(2vw + 10px);
/* 2% от ширины + фиксированные 10px */
```

## ФИНАЛЬНЫЕ ШАГИ

1. **Создай файл `adaptive.css`**
2. **Подключи его в каждом HTML файле**
3. **Добавь бургер-меню в HTML (кнопка с иконкой)**
4. **Тестируй адаптацию**:
   - Открой Dev Tools (F12)
   - Включи Responsive Design Mode
   - Тестируй от 1920px → 1240px → 768px → 360px → 320px
5. **Проверь все брейкпоинты по чеклисту**
6. **Подстрой значения `clamp()` если нужно**
